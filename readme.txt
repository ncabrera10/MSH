MSH implementation for the VRPTR without carpooling.
Number of instances: 40
-------------------------------------------------------------------------------------------------------------
If you want to use this code:
1. Check that all the instances files from Coindreau et al. (2019) in txt format are on the "instances" folder.
2. Open the class "MVC" in the "main" package.
3. Run as a Java Application.

All your results and reports will be store on the "results" folder.

----------------------------------------------------------------------------------------------------------------

For plotting your solutions go to: https://nicolascabrera.shinyapps.io/VRPTR/
